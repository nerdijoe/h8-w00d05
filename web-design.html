Get to Know Adaptive Web Design (AWD) dan Responsive Web Design (RWB)

Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Adaptive Web Design uses multiple page layouts for a single webpage. The Specific page will be loaded a page version that is best suited for user's device.

Responsive Web Design uses only 1 page layout that will change fluidly to various screen size dimension.

Both designs have pro and cons.

Adaptive Web Design is useful when we want to specifically target specific device or screen size because developer have more control to design specific layout and content. However, there will extra work to design and develop a page for multiple devices.

On the other hand, the responsive web design will be easier to build and maintain. However the responds time is slower than the adaptive web design.

Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Yes, the smartphone and tablet will change the website design. Because of the growing usage of mobile devices, companies will create sites with the mobile specific design layout and content.

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Adaptive Web Design could help people with dissabilities using the web. For example, for color-blind people, we could make a special page with different color so that they can read the site.
